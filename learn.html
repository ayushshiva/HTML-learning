Definition

What is HTML?

HTML = Hyper Text Markup Language.

It is the code used to structure a web page and its content.

The components used to design the structure of websites are called HTML tags.

First Page

First HTML File

Common first file name is index.html.

It is usually the default name for a websiteâ€™s homepage.

Tag vs Element

HTML Tag & Element

An HTML tag is a container for some content or other HTML tags.

<p> This is a paragraph </p>

Element = opening tag + content + closing tag.

Syntax

Basic HTML Page

<!DOCTYPE html> <!-- tells browser you are using HTML5 -->
<html> <!-- root of the document -->
<head> <!-- container for metadata -->
<title>My First Page</title> <!-- page title -->
</head>
<body> <!-- contains all data rendered by browser -->
<p>hello world</p> <!-- paragraph tag -->
</body>
</html>
Quick Points

HTML Basics â€“ Recap

  • <html> tag is parent of <head> & <body>.
  • Most HTML elements have opening & closing tags with content in between.
  • Some tags have no closing tag (void tags), e.g. <br>, <hr>, <img>.
  • You can use Inspect Element / View Page Source in browser to see HTML.

Comments in HTML

Note

HTML is NOT Case Sensitive

<head> = <HEAD>
